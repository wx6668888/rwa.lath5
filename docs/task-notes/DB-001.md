## 环境验证边界

当前工作机未安装 Docker、`psql` 或本地 PostgreSQL 服务，因此任务保持“进行中”，不提前进入“待审核”。迁移的真实 `run → revert → run` 循环尚未在本机执行；命令连接 `127.0.0.1:5432` 时得到预期的 `ECONNREFUSED`。审核前需在具备 Docker 的环境执行：

1. `docker compose up -d postgres`
2. `npm run db:migration:run`
3. `npm run db:migration:revert`
4. `npm run db:migration:run`
5. 分别以开发和测试 URL 查询 `schema_migrations`，确认两库互不影响。
